# How to Use

  Clone the project:
```
  $ git clone https://github.com/junyorff12/react-calc.git
```
  Enter the project directory
```
  $ cd react-calc/
```
  Install dependencies
```
  $ yarn install
```
  Run the project (it will open at :3000 port by default)
```
  $ yarn start
```
